Anotações módulo de links

Conhecendo a tag <a>

  • href é o atributo que vai receber um link de uma página para redirecionar
  • Target é o atributo responsável por trocar a aba ou redirecionar na mesma aba
  • self => abre o link na mesma aba
  • blank => redireciona o link para outra aba

Utilizando a tag <a>

  • Sites ou Links <a href="#https://rocketseat.com.br" target="_blank"></a>
  • Email <a href="mailto:gabrielserq@gmail.com" target="_blank"></a>
  • Telefone <a href="tel:+5511984563230"></a>
  • Posso envolver essa tag em qualquer outra tag

Como Navegar pelos caminhos

  • Mesmo diretório <a href="./urls-caminho.html" target="_blank"></a>
  • Saindo de diretórios <a href="../urls-caminho.html" target="_blank"></a>

URLs absolutos x relativos

  • Absoluto - inclui o protocolo e nome do domínio e sempre apontará pra o mesmo local, pois é absoluto
  • Relativo - relativo página aberta no momento, apontará para lugares diferentes
