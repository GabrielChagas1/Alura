Anotações módulo de semântica

Aula - Semântica

  • Importante para a indexação do seu site no google
  • Importante para pessoas com deficiências visuais
  • Se perguntar qual o melhor elemento que pode ser usado para construir o que eu estou pensando

Aula - Títulos e paragráfos

  • para estilizar os títulos temos seis tipos de separações, que começa do h1 até o h6
  • Para estilizar os paragráfos usamos a tag <p>conteúdo</p>

Aula - Listas

  • Existem dois tipos de listas - Ordenadas e não Ordenadas
  • Não Ordenadas - <ul> conteúdo </ul>
  • Ordenadas - <ol>conteúdo</ol>

Aula - Citações

  • Para citações usamos as tags <blockquote> </blockquote>
  • dentro da tag temos que usar o atributo cite que leva o site da onde tiramos nossa citação

Aula - Abreviações

  • Para abreviações usamos a tag <abbr> junto com o atributo title para quando o usuário descansar o mouse sobre a palavra, irá aparecer o texto que está dentro do title

Aula - Contato

  • tag para guardar informações de quem desenvolveu o site
  • <address></address>

Aula - Lista de Descrição

  • tag usada para criar uma lista de descrições semelhante a um glossário
  • Dentro de cada <dl></dl> temos que ter o </dt> que o título e temos também o </dd> que é a descrição da palavra que está dentro da </dt>

Aula - Representação de código

  • Para partes genéricas de código usamos a tag <code>
  • Para blocos de códigos, usamos a tag <pre> junto com a tag <code> porque essa tag mantém os espaços em brancos e recuos que eu colocar no meu código

Aula - Elementos Genéricos

  • Temos duas tags que são genéricas
  • <div> faz uma quebra na linha
  • <span> Não quebra a linha dos elementos
