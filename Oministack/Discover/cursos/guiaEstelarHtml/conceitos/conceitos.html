Anotações módulo de conceitos

Aula - O que é HTML

  • Acrônimo para HyperText Markup Language (Linguagem de marcação de texto).

Aula - Anatomia das Tags

  • Elementos Vazios não precisam serem fechadas (img, input)

Aula - Atributos

  • Atributos são informações extras ou configurações de alguma tag
  • a tag img possui src="" e o alt=""
  • Atributos booleanos não precisam de informações extra
  • Não se deve omitir ou usar aspas simples, opte sempre por aspas duplas

Aula - Atributos Globais

    Atributos Globais mais utilizados

  • class - atributos para adicionar uma classe a diversos elemento
  • contenteditable - atributo para tornar um elemento editável
  • data-* - atributo para expandir novos atributos para fazer lógica com o javascript
  • hidden - atributo para esconder o conteúdo
  • id - atributo para adicionar um valor único aos meus elementos
  • style - aplicar estilos css para o próprio elemento
  • tabindex - serve para criar uma ordem que o tab deve seguir para navegar
  • title - serve para definir um título, usando muito nas tags </img>
